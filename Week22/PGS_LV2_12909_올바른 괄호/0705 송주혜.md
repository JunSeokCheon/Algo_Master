```
1. 괄호를 담을 스택 생성
2. "(" 인 경우 스택에 append
3. ") 인 경우 만약 스택이 비어있지 않다면 스택의 제일 처음이 "(" 일 때 pop
4. 스택이 비어있으면 True, 아니면 False
```

```python
def solution(s):
    answer = ""
    stack = []

    for i in s:
        if i == "(":
            stack.append(i)
        elif stack and i == ")":  # )인경우
            if stack[-1] == "(":
                stack.pop()
        else:
            return False

    if not stack:
        answer = True
    else:
        answer = False

    return answer


print(solution(")()("))
```
