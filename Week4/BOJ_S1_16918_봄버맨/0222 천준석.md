<접근법>
``` 
0. 초기 상태와 1초가 같다는 것을 파악하고, 폭탄이 터진 조건(3초 전)을 설정하고, 그 외의 조건은 모든 칸이 폭탄이 가진다고 설정한다.
1. 폭탄 폭발 함수로는 bfs을 활용하여 상하좌우 뻗으면서 조건에 따라 'O' -> '.' 로 수정한다.
2. time = 1일 때 초기 값이랑 같으니 그대로 값을 받지만 폭탄 위치를 deque에 저장한다. (폭발을 위해)
3. 3초가 지난 시점일때 bfs(함수 폭발 함수)를 실행하고, bfs 함수가 실행되면 또 폭탄이 생기기 때문에 바로 폭탄 위치를 deque에 저장한다.
4. 그 외는 모든 칸이 폭탄을 가지고 있다고 가정하니깐 모든 칸을 'O'로 설정
*** 힌트를 보면서 격자판의 상태를 이해하면 문제 접근하기 용이하다.***

```

```python
import sys
from collections import deque

R, C, N = map(int, sys.stdin.readline().split())
board = []
que = deque()
dx = [0, 0, -1, 1]
dy = [1, -1, 0, 0]

for _ in range(R):
    board.append(list(map(str, sys.stdin.readline().strip())))

# 상하좌우 폭탄 폭발 함수
def bfs(que, board):
    while que:
        x, y = que.popleft()
        board[x][y] = '.'
        
        for i in range(4):
            nx = dx[i] + x
            ny = dy[i] + y

            if nx < 0 or nx >= R or ny < 0 or ny >= C:
                continue
            
            if board[nx][ny] == 'O':
                board[nx][ny] = '.'


def bomb(time):
    global que, board
    # 초기 1초에 폭탄 위치 deque에 저장
    if time == 1:
        for i in range(R):
            for j in range(C):
                if board[i][j] == 'O':
                    que.append((i,j))
    # 3초, 5초, 7초 ... -> 폭탄 터질 시간
    elif time % 2 == 1:
        # 3초가 지난 폭탄 폭발
        bfs(que, board)

        # 3초가 지나서 폭탄이 폭발한 뒤 생긴 폭탄을 deque에 저장
        for i in range(R):
            for j in range(C):
                if board[i][j] == 'O':
                    que.append((i,j))
    # 모든 칸이 폭탄을 가지고 있는 상태
    else:
        board = [['O']*C for _ in range(R)]

for time in range(1, N+1):
    bomb(time)

for result in board:
    print(''.join(result))
```
