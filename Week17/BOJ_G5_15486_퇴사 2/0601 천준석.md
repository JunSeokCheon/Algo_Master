# BOJ_G5_15486_퇴사 2 [2023-06-01]
https://www.acmicpc.net/problem/15486

<접근법>
``` 
0. 해당 날짜에 상담하는 경우와 상담을 하지 않는 경우로 나눌수 있다.
1. 상담한다면, 상담이 끝난 다음 날 수익이 그 전날 값을 더해 증가한다.
2. 상담을 하지 않는다면(못) 다음날로 넘어간다.
3. 값을 갱신할 때, "현재까지의 수익 + 이번 상담의 수익(dp[i-1]+p) VS 오늘의 상담이 끝나는 시점의 수익(dp[i+t-1] = t일을 더하고 난 뒤 받을 수입)"을 비교하여 최대값 갱신한다.
```


```python
import sys

n = int(sys.stdin.readline())
dp = [0] * (n+1)

for i in range(1, n+1):
    t, p = map(int, sys.stdin.readline().split())
    # dp 배열 갱신
    dp[i] = max(dp[i], dp[i-1])
    # 상담 종료일이 n일차를 벗어나다면 조건에 맞지 않는다.
    # n이 아니고 n+1인 이유는 n일차에서도 최대 1일까지 상담이 존재한다. (무조건 1일의 상담은 있기 때문에)
    if i + t <= n + 1:
        # 오늘의 상담이 끝나는 시점의 수익과 현재까지의 수익에 이번 상담의 수익을 더한 값 중 큰 값을 저장
        dp[i+t-1] = max(dp[i+t-1], dp[i-1]+p)
        
print(dp[n])
```
