```
<접근법>
1. 퀸은 대각선 / 좌우로 이동가능
2. 1차원 배열을 2차원 배열처럼 사용하는게 포인트
```
```python
import sys
input = sys.stdin.readline
n= int(input())
ans=0
row = [0]*n

def check(x):
    for i in range(x):
        if row[x]==row[i] or abs(row[x]-row[i])==abs(x-i): #열이 같거나 대각선
            return False
    return True
            
def back(x):
    global ans
    if x==n:
        ans+=1
        return
    else:
        for i in range(n):
            row[x]=i #[x,i]에 퀸 두기
            if check(x):
                back(x+1)
back(0)
print(ans)
```
