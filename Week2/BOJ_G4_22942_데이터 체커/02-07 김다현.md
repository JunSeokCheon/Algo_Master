<접근법>
```
1. 인접한 원들끼리 비교해야하기때문에 중심점을 기준으로 sort
2. 끝점이 닫히기 전에 다른 원의 시작점이 들어오면 겹친다고 판단
```
```python
'''
메모리 : 78500
시간 : 560
'''
import sys
n=int(input())
circle=[]
stack=[]
for num, i in enumerate(range(n)):
    c, r = sys.stdin.readline().split()
    circle.append((int(c)-int(r),i,0)) #처음이면 0으로 저장
    circle.append((int(c)+int(r), i,1)) #마지막이면 1로 저장
circle.sort()

for i in range(n):
    check = circle[i][2]
    if check==0: #원의 시작이면
        stack.append(circle[i])
    else: 
        if stack[-1][2]==0: #전 원이 아직 닫히지 않았다면,
            if stack[-1][1] == circle[i][1]: #닫히지 않은 원 == 닫으려는 원
                stack.pop()
            else:
                print('NO')
                exit(0) #return과 같은 것 
print('YES')
```
