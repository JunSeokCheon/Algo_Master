```
1. 인덱스 부여, 최소값인지 최대값인지 부여
2. 같으면 끝, 스택에 있는 인덱스 다르면 끝

```

```python


import heapq
n= int(input())
#heap만들고
#
heap=[]
for _ in range(n):
    nums = map(int,input().split())
    
    for num in nums:
        if len(heap) < n:
            heapq.heappush(heap,num)
            
        else:
            if heap[0] < num:
                heapq.heappop(heap)
                heapq.heappush(heap,num)
                
print(heap[0])     
```
